# FlagQuiz

> A SwiftUI App using custom views and adding a simple animation.

[![Swift Version][swift-image]][swift-url]
[![License][license-image]][license-url]

Exploring SwiftUI with a simple flag quiz game.

1. SwiftUI
2. Alerts

4. Gradients
5. Animation
